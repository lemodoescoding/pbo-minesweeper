### Prerequisite

- Java Installed
- Java Compiler Installed
- JDBC (MySQL Connector Installed)
- MySQL Server installed (you can use XAMPP)

### Setup for Database

1. Open up `./ddl.sql`.
2. Run it on your MySQL Server.
3. Open up `./Minesweeper/src/Database.java`, and update the appropriate data for your username and password (MySQL)

### How to run

1. go to the `./Minesweeper/src/`
2. Compile the entire `src` folder
3. Run using `java -cp .:/path/to/file/java/mysql-connector-java.jar App`, make sure to adjust the file path to the .jar connector file
